# Chunjae
Spring Boot로 게시판 만들기

# Page Flow
1. 회원가입
2. 로그인 (HttpSession)
3. 글쓰기
4. 글 목록보기 (page 처리) / http://localhost:8080/ (첫 url 주소)
5. 글 상세보기
6. 글 수정하기
7. 글 삭제하기
8. 로그아웃
9. 관리자 - 어떤 사용자의 글이든 삭제할 수 있도록.

# technology Used
1. Spring Boot
2. Spring MVC - 웹
3. Thymeleaf - 웹 관련 템플릿 엔진
4. Spring JDBC
5. MSSQL
6. Lombok (dto)
